Datos del software.
------------------

- El programa calcula los rangos de los tamaños para un set de datos determinado.
- Los datos deben ser ingresados en el formulario separados por comas (,). El separador para decimales
es el punto (.).
- Como resultado se obtendrán los rangos para los diferentes tamaños de las partes de software:
  + VS Very Small
  + S  Small
  + M  Medium
  + L  Large
  + VL Very Large
